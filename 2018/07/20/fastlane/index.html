fastlane自动化打包&自动上传pgyer&钉钉机器人自动发布

发表于 2018-07-20 | 分类于 tool |

fastlane is a tool for iOS and Android developers to automate tedious tasks like generating screenshots, dealing with provisioning profiles, and releasing your application.

Github

官网 文档

为什么选择fastlane

  • 配置好之后一行代码直接实现打包， 上传到分发提高效率

开始操作

  • 安装最新版的Xcode命令行工具：

xcode-select –install

  • 安装fastlane

[sudo] gem install fastlane -NV
或者brew cask install fastlane 选择适合自己的安装方式

  • 初始化fastlane

cd 到项目目录下 并执行 fastlane init

如图会出现四个选项，根据自己的情况进行选择，这里我选择3

接下来按要求输入开发者账号和密码。

打包到pgyer

pgyer在fastlane是作为一个插件存在的，所以要打包到pgyer必须先安装pgyer的插件，终端输入：

fastlane add_plugin pgyer

执行

在工程目录下，点击Go2Shell来到终端的工程路径下，输入

fastlane debug

等待执行完成，视项目大小，打包上传时间在几分钟到十几分钟不等。

关于fastlane的一些想法

其实对于很多小团队来说，fastlane就可以简化很多操作，提升一些效率，但是还不够极致，因为我们没有打通Git环节，测试环节，反馈环节等，fastlane只是处于开发中的一环。许多团队在进行Jenkins或是其他的CI的尝试来摸索适合自己的工作流。但是也不要盲目跟风，从需求出发切合实际就好，找到痛点才能找到止痛药！
