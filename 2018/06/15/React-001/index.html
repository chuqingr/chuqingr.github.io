React Native 学习笔记

发表于 2018-06-15 | 分类于 iOS |

UI–Flex布局

一.语法

1.容器的属性（容器类似父视图既父视图的属性，决定的是内部的子视图的排列方式）

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content
1.flex-direction属性
.box {
   flex-direction: row | row-reverse | column | column-reverse;
}

ta有四个值：

  • row：轴为水平方向，起点在左（默认）
  • row-reverse：轴为水平方向，起点在右
  • column：轴为垂直方向，起点在上
  • column-reverse：轴为垂直方向，起点在下
2.flex-wrap属性
.box{
      flex-wrap: nowrap | wrap | wrap-reverse;
}

ta有三个值：

  • nowrap：不换行（默认）
  • wrap：换行，第一行在上
  • wrap-reverse：换行，第一行在下
3.flex-flow属性
.box {
      flex-flow: <flex-direction> || <flex-wrap>;
}

flex-flow属性是flex-direction属性和flex-wrap属性的简写形式，默认值为均为默认值（row && nowrap）

4.justify-content属性定义了项目在主轴上的对齐方式
.box {
      justify-content: flex-start | flex-end | center | space-between | space-around;
}

ta有五个值：

  • flex-start：左对齐（默认值）
  • flex-end：右对齐
  • center： 居中
  • space-between：两端对齐，项目之间的间隔都相等。
  • space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。
5.align-items属性定义项目在交叉轴上如何对齐
.box {
      align-items: flex-start | flex-end | center | baseline | stretch;
}

ta有5个值，具体的对齐方式与交叉轴的方向有关，下面假设交叉轴从上到下

  • flex-start：交叉轴的起点对齐
  • flex-end：交叉轴的终点对齐
  • center：交叉轴的中点对齐
  • baseline: 项目的第一行文字的基线对齐
  • stretch：如果项目未设置高度或设为auto，将占满整个容器的高度（默认值）
6.align-content属性定义了多根轴线的对齐方式。如果项目只有一根轴线，该属性不起作用。
.box {
      align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}

ta有6个值：

  • flex-start：与交叉轴的起点对齐
  • flex-end：与交叉轴的终点对齐
  • center：与交叉轴的中点对齐
  • space-between：与交叉轴两端对齐，轴线之间的间隔平均分布
  • space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍
  • stretch：轴线占满整个交叉轴（默认值）

2.项目的属性（项目，类似子视图既子视图与子视图之间的排列方式|优先级）

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self
1.order属性定义项目的排列顺序。数值越小，排列越靠前，默认为0

2.flex-grow属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大

3.flex-shrink属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小
4.flex-basis属性定义了在分配多余空间之前，项目占据的主轴空间（main size）。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。

它可以设为跟width或height属性一样的值（比如350px），则项目将占据固定空间。

5.flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。

该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。
建议优先使用这个属性，而不是单独写三个分离的属性，因为浏览器会推算相关值。

6.align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

该属性可能取6个值，除了auto，其他都与align-items属性完全一致。
