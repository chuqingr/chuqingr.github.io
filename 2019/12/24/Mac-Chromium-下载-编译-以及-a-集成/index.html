Mac Chromium 下载 & 编译 以及.a 集成

发表于 2019-12-24 |

下载

  • 1.首先 终端需要翻墙

  • 2.下载 chromium 提供的工具，depot_tools

    git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git

  • 3.将 depot_tools 的目录添加到环境变量

    export PATH=“$PATH:_path_depot_tools” | path 为depot_tools文件的路径

  • 4.新建一个 文件夹 & 打开文件夹

    $ mkdir chromium && cd chromium

  • 5.开始下载

    $ fetch –no-history chromium | –no-history 的意思是不下载历史版本的信息

  • 6.等待下载完成，（10M的速度大概需要20分钟左右能下载完成）

编译

现在chromium 使用 GN 来编译构建

  • 1.进入 src 目录，输入

    gn gen out/default | gen 命令会根据当前的 目录下的 .gn 以及指向的BUILDCONFIG.gn && BUILD.gn 等文件 ，生成对应的 .ninja 文件，真正进行编译的 是 .ninja 文件

  • 2.执行

    ninja -C out/defalut 命令，就会开始漫长的编译过程，根据机器性能，编译过程大概在 5 - 10 小时不等。大约要编译4w+的文件.
